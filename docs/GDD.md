# Rocket Ride — Game Design Document

## Концепция
Crash-hybrid iGaming: ракета взлетает, множитель растёт (e^0.003t), cash out до краша. 5 механик + 4 bet modes.

## Flow
```
[IDLE] → Bet → [3..2..1] → [FLYING] → CASH OUT (win) или CRASH (loss)
```

## Монетизация
House edge 4%. 100K × $10 avg = $40K GGR.
